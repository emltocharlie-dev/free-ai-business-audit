# 10xclaw Brand v1 (效率 / 创新 / 可信)

This set is generated following the brand-identity constraints:
- **Efficiency**: clear geometry, fast recognition, minimal clutter
- **Innovation**: teal→cyan→sky palette, tech-forward gradients
- **Trust**: audit ring + check accent, works in monochrome

## Files
- `10xclaw-brand-v1-icon.svg` — primary icon (ring + X + subtle check)
- `10xclaw-brand-v1-favicon.svg` — simplified icon for very small sizes
- `10xclaw-brand-v1-wordmark.svg` — path-only wordmark
- `10xclaw-brand-v1-lockup-horizontal.svg` — icon + wordmark for website headers

## Usage
- Website header: use `lockup-horizontal.svg`
- Favicon/app icon: use `favicon.svg` (then export PNG/ICO)
- Dark background: prefer icon/wordmark with lighter gradient; for strict monochrome use a white override.
